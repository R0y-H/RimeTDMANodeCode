Contiki系统的核心程序：内核/网络协议/文件系统/库/设备/动态加载/GUI

文件名                    主要内容
cfs        The Contiki file system interface
ctk        The Contiki Toolkit (CTK) provides the GUI
dev        Devices of sensor node based on the Contiki
lib        Library as: list/memory/random/crc/assert/...
loader     Dynamic linker and loader based on the Contiki
net        Network protocol as: mac/rime/rpl/uip
sys        Kernel of the Contiki as: process/etimer/rtimer/ctimer/...
